# Page Override: Agriculture Sector

## Purpose

Track farms, crops, seasons, production, expenses, sales, and season profitability.

## Primary user question

Which crop season made money, lost money, or needs intervention?

## Required records

- Farm
- Crop
- Season
- Input expense
- Labor expense
- Production batch
- Sale
- Loss/waste record

## KPI cards

- المواسم النشطة
- تكلفة الموسم
- كمية الإنتاج
- المبيعات
- صافي ربح الموسم
- الهدر / الفاقد

## Table columns

- المزرعة
- المحصول
- الموسم
- تاريخ البداية
- تكلفة المدخلات
- الإنتاج
- المبيعات
- الربحية
- الحالة

## Design notes

Use agriculture green only as a sector marker. Financial values still use the global finance color rules. Do not make the agriculture section decorative or cartoonish.
