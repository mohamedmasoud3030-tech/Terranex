# Page Override: Real Estate Sector

## Purpose

Manage lands, properties, acquisition, development costs, sales, valuations, documents, and asset profitability.

## Primary user question

Which real estate assets are profitable, under development, blocked, or ready for sale?

## Required records

- Asset / property
- Acquisition transaction
- Development cost
- Sale transaction
- Valuation
- Ownership document
- Partner share

## KPI cards

- عدد الأصول
- تكلفة الشراء
- تكلفة التطوير
- قيمة البيع / التقييم
- صافي الربح / الخسارة
- أصول تحت التطوير

## Table columns

- الأصل
- النوع
- الموقع
- الحالة
- تكلفة الشراء
- تكلفة التطوير
- التقييم الحالي
- الربح / الخسارة
- المستندات

## Design notes

Use trust navy with restrained gold accents. Avoid luxury-brochure layouts inside the operating app. Photos are secondary; records and numbers are primary.
