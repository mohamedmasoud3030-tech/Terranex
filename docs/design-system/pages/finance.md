# Page Override: Finance and Obligations

## Purpose

Unify income, expenses, partner obligations, receivables, payables, profit/loss, and audit trails across all sectors.

## Primary user question

Who owes money, who should be paid, and what is the financial truth by sector/project/period?

## Required records

- Transaction
- Account
- Partner
- Receivable
- Payable
- Allocation to sector/project/asset/season/herd
- Attachment/document
- Approval or decision

## KPI cards

- إجمالي الإيرادات
- إجمالي المصروفات
- صافي الربح / الخسارة
- الذمم المدينة
- الذمم الدائنة
- النقد المتاح
- بنود غير مصنفة

## Table columns

- التاريخ
- النوع
- القطاع
- المشروع / الأصل / الموسم / القطيع
- الطرف
- المبلغ
- الحالة
- المستند
- أضيف بواسطة

## Design notes

This page is the strictest page. Use Financial Dashboard style, visible totals, audit trail table, clear filters, and consistent currency formatting.
