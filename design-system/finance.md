# Finance
## Layout rules
- Ledger control center with strict transaction table.
## Components needed
- KPI cards, filters matrix, transaction table, reconciliation panel.
## KPI cards
- Income period, expense period, net cash movement, open obligations.
## Tables
- Date, direction, amount, category, counterparty, project, document, status.
## Filters
- Direction, category, project, partner, document state, date range.
## Forms
- Transaction entry, reconciliation action, category management.
## Badges
- Income, expense, linked doc, missing doc, reconciled.
## Charts
- Cash in/out trend.
## Empty states
- "No transactions in selected period."
## Accessibility checklist
- Table row actions keyboard accessible.
## RTL rules
- Amount column fixed-width for quick scan.
## Anti-patterns to avoid
- Editable free-text categories without governance.
