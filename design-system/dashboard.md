# Dashboard
## Layout rules
- Executive header, KPI row, trend charts, obligations table.
## Components needed
- KPI cards, alert strip, line chart, receivables/payables split, recent activity table.
## KPI cards
- Total portfolio value, net profit YTD, receivables, payables.
## Tables
- "Critical obligations" with due date, counterparty, amount, project.
## Filters
- Date range, sector, project status.
## Forms
- Quick add transaction modal (amount, direction, project, document ref).
## Badges
- Due today, overdue, under review, settled.
## Charts
- Profit trend line and sector profit bar.
## Empty states
- "No financial activity yet" + CTA to add first transaction.
## Accessibility checklist
- Chart data table fallback, clear tab order, focus rings.
## RTL rules
- KPI order starts from right; numeric figures aligned left within cards.
## Anti-patterns to avoid
- More than 2 chart cards above fold.
