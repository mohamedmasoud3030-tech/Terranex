# Projects
## Layout rules
- Page-level summary band + primary projects table.
## Components needed
- KPI cards, filters row, project table, right panel summary.
## KPI cards
- Active projects, on-hold projects, total invested, average margin.
## Tables
- Columns: name, sector, capital, revenue, net, status, manager.
## Filters
- Sector, profitability status, manager, last update.
## Forms
- New project form with sector-specific template selector.
## Badges
- Active, paused, closed, high risk, profitable.
## Charts
- Optional bubble chart: investment size vs margin.
## Empty states
- Explain project modeling and provide "Create Project".
## Accessibility checklist
- Sort controls keyboard reachable, table headers announced.
## RTL rules
- Sticky action column on left edge.
## Anti-patterns to avoid
- Unstructured project cards replacing auditable table.
