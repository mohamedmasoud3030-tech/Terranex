# Reports
## Layout rules
- Report selector + output view.
## Components needed
- KPI strip, report presets, preview table, export bar.
## KPI cards
- Current net profit, sector best performer, worst performer, variance.
## Tables
- Project P/L matrix and obligations report.
## Filters
- Period, sector, project, partner, status.
## Forms
- Save report view preset.
## Badges
- Final, draft, verified, requires review.
## Charts
- Comparison bars and trend lines by period.
## Empty states
- "No report data for this filter."
## Accessibility checklist
- Export actions labeled, table summaries.
## RTL rules
- Column titles concise Arabic finance terms.
## Anti-patterns to avoid
- Beautiful charts without source table.
