# Agriculture
## Layout rules
- Seasons overview + operational and financial detail.
## Components needed
- Season KPI cards, crop table, input cost panel, production chart.
## KPI cards
- Active seasons, yield estimate, season cost, expected profit.
## Tables
- Farm, crop, season window, cost, expected revenue, status.
## Filters
- Crop type, farm, season phase, profitability.
## Forms
- Create season, log input expense, harvest entry.
## Badges
- Planting, growing, harvest, closed.
## Charts
- Yield vs cost per season.
## Empty states
- "No agriculture seasons yet."
## Accessibility checklist
- Form grouping, keyboard filters, readable table density.
## RTL rules
- Date intervals displayed in clear Arabic labels.
## Anti-patterns to avoid
- Seasonal data mixed without year boundaries.
