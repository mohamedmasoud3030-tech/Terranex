# Project Detail
## Layout rules
- Overview at top, tabbed sections below.
## Components needed
- Header summary, KPI strip, tabs, transactions table, docs list, task timeline.
## KPI cards
- Invested, earned, net, unpaid obligations.
## Tables
- Ledger-like transaction table with document links.
## Filters
- Transaction category, date, direction, counterparty.
## Forms
- Add expense/income, attach document, register obligation.
## Badges
- Draft, verified, disputed, settled.
## Charts
- Cumulative cashflow line chart.
## Empty states
- No transactions/document/task placeholders with next action.
## Accessibility checklist
- Tab semantics, form error messaging, screen-reader labels.
## RTL rules
- Timeline marker anchored right.
## Anti-patterns to avoid
- Hidden financial formula assumptions.
