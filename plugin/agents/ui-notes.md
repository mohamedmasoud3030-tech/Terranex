# UI Notes

This document tracks UI and UX considerations for Terranex.

## Main screens

- dashboard
- project overview
- project profitability
- transactions
- obligations
- receivables and payables
- partner balances
- documents
- reports

## Design direction

- operational clarity
- dense but readable information
- strong hierarchy
- traceable financial context
- quick navigation
- desktop-first workflows

## Important concerns

- visibility of financial status
- clear relationship between projects and transactions
- document linkage
- timeline readability
- low-friction operational forms
