# Understand Domain

This skill extracts and organizes Terranex business knowledge.

## Focus areas

- sectors
- projects
- assets
- partners
- transactions
- obligations
- documents
- profitability

## Expected outputs

- domain summaries
- business flow descriptions
- relationship maps
- missing entity analysis
- implementation gaps
- operational concerns

## Main question

The resulting understanding should help answer:

- Did the project make profit or loss?
- Who owes money?
- Which records support the answer?
